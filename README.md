# 网页数据导出器 - 谷歌浏览器插件

这是一个简单易用的谷歌浏览器插件，可以帮助您快速打开指定URL，并将页面中的数据（如表格、列表、段落）导出为CSV文件，方便进行数据分析和处理。

## 功能特点

- ✨ 快速打开指定的URL
- 📊 自动提取页面中的表格数据
- 📋 智能识别列表内容
- 📝 提取重要段落信息
- 💾 将提取的数据导出为CSV文件
- 💾 记住您的URL和文件名设置
- 🎨 简洁直观的用户界面

## 安装说明

### 开发者模式安装（推荐）

1. 确保您已下载本插件的所有文件，保存在本地目录中（如 `D:\work\WebScraper`）
2. 打开谷歌浏览器，在地址栏输入 `chrome://extensions/` 并按回车
3. 开启页面右上角的「开发者模式」开关
4. 点击「加载已解压的扩展程序」按钮
5. 选择您保存插件文件的目录（如 `D:\work\WebScraper`）
6. 插件将被安装，您可以在浏览器右上角看到插件图标

### 使用注意事项

- 确保您的浏览器版本支持Chrome扩展Manifest V3
- 插件需要访问标签页和下载文件的权限
- 对于某些受保护的网站，可能无法提取数据

## 使用方法

### 方法一：打开URL并提取数据

1. 点击浏览器右上角的插件图标，打开插件界面
2. 在「输入URL」文本框中输入您想要分析的网页地址
3. 点击「打开URL」按钮，插件将在新标签页中打开该URL
4. 等待页面加载完成后，点击「提取当前页面数据」按钮
5. 插件会自动分析页面并提取可用数据
6. 在「导出文件名」输入框中设置文件名（可选，默认为"export"）
7. 点击「导出为CSV」按钮，数据将下载为CSV文件

### 方法二：直接提取当前页面数据

1. 导航到您想要分析的网页
2. 点击浏览器右上角的插件图标，打开插件界面
3. 直接点击「提取当前页面数据」按钮
4. 设置导出文件名并点击「导出为CSV」按钮

## 数据提取规则

插件会按照以下优先级提取数据：

1. **表格数据**：自动识别页面中的所有`<table>`元素，提取表头和数据行
2. **列表数据**：如果页面中没有表格，会尝试提取`<ul>`和`<ol>`列表
3. **段落数据**：如果以上都没有，会提取页面中的主要段落文本

## 技术说明

- 使用Chrome扩展Manifest V3标准开发
- 采用JavaScript进行数据提取和处理
- 使用Chrome API进行标签页管理和文件下载
- 支持UTF-8编码，确保中文和其他特殊字符正确显示

## 常见问题

### Q: 为什么某些网站无法提取数据？
A: 某些网站可能使用了复杂的JavaScript渲染或反爬虫措施，可能导致数据提取失败。

### Q: 导出的CSV文件中文显示乱码怎么办？
A: 请使用Excel、Notepad++等支持UTF-8编码的软件打开，或在导入Excel时选择UTF-8编码。

### Q: 如何自定义数据提取规则？
A: 您可以修改 `content.js` 文件中的 `extractPageData()` 函数来自定义提取逻辑。
